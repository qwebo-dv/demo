# ArchTerra

Премиальный лендинг для ООО «АрхТерра Групп» / бренда ArchTerra.

## Технологии

- Next.js App Router
- TypeScript
- Адаптивная CSS-вёрстка без внешних изображений: визуальная атмосфера собрана градиентами и фактурами, чтобы стартовая загрузка была быстрой.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
