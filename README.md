# Интернет-магазин сантехники на Payload CMS

Готовый стартовый шаблон backend-части интернет-магазина сантехники на базе **Payload CMS + PostgreSQL**.

## Что уже настроено

- Аутентификация и роли пользователей (`admin`, `customer`).
- Категории товаров с иерархией.
- Каталог товаров: артикул, бренд, цена, остаток, изображения, характеристики.
- Заказы: позиции, снимок цены, статусы, адрес доставки.
- Глобальные настройки магазина (контакты, доставка, оплата).
- Коллекция медиа для изображений товаров.

## Структура данных

### Коллекции

- `users` — администраторы и клиенты.
- `categories` — категории (например: Смесители, Ванны, Инсталляции).
- `products` — товары сантехники.
- `orders` — заказы покупателей.
- `media` — изображения.

### Global

- `store-settings` — название магазина, контакты, условия доставки и оплаты.

## Быстрый старт

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Запустите PostgreSQL:

   ```bash
   docker compose up -d
   ```

3. Создайте `.env` на основе примера:

   ```bash
   cp .env.example .env
   ```

4. Запустите Payload в режиме разработки:

   ```bash
   npm run dev
   ```

5. Откройте админку:

   - `http://localhost:3000/admin`

## Что сделать следующим шагом

- Подключить фронтенд (Next.js/Nuxt) к REST/GraphQL API Payload.
- Добавить фильтры по бренду, цене, диаметру подключения, типу монтажа.
- Реализовать интеграции оплаты (ЮKassa / T-Bank) и доставки (СДЭК / Boxberry).
- Настроить SEO-поля для категорий и карточек товаров.
- Добавить промокоды и остатки по складам.

## Команды

- `npm run dev` — запуск в dev-режиме.
- `npm run build` — production build.
- `npm run start` — запуск production-сборки.
- `npm run generate:types` — генерация `payload-types.ts`.
